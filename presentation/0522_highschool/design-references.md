# Design References Used

Lazyweb MCP로 다음 패턴을 확인하고 슬라이드에 반영했다.

- Microsoft AI in Education — 밝은 교육 톤, hero + card grid 구조
- Apple Education — 교육/성장 스토리텔링, 큰 이미지/짧은 문장 리듬
- GitHub Spark / Genie / Speechify — AI 파트에 어울리는 dark tech keynote 분위기
- Canva Design School / Brainscape — 학생 대상 학습 카드, 친근한 교육 허브 느낌
- Max Batt AI Report Card — 한 문장만 크게 보여주는 minimalist title slide

실제 슬라이드에는 외부 스크린샷을 삽입하지 않고, 위 패턴을 참고해 CSS/SVG/HTML 형태로 자체 제작했다.
